Subscribes user on JustClick mailing list after registration

== Description ==

Use this plugin, if you want to subscribe registered users to you mailing lists on JustClick website.
PHP 5 is required.
Find more information on [JustClick website](http://www.justclick.ru/)

Используйте данный плагин для того, чтобы подписывать на ваши рассылки в системе JustClick пользователей, регистрирующихся на вашем сайте.
Для работы плагины необходимо использовать PHP 5.
Дополнительная информация [на сайте JustClick](http://www.justclick.ru/)

== Installation ==

1. Upload `justclick-subscriber` directory to the `/wp-content/plugins/` directory
1. Activate the JustClick Subscriber plugin through the «Plugins» menu in WordPress
1. New item will appear in the Options menu
1. Check «Subscribe user after register» and fill in JustClick subscribe form code on the plugin settings page
1. Try to register

---

2. Загрузите директорию `justclick-subscriber` в директорию `/wp-content/plugins/` на вашем сайте
2. Активируйте плагин в меню «Плагины»
2. В меню «Параметры» появится новый пункт «Подписка JustClick»
2. Отметьте «Включить подписку при регистрации» и скопируйте код формы подписки в соответвтвующее поле
2. Попробуйте зарегистрироваться
