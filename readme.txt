Subscribes user on JustClick mailing list after registration

== Description ==

Use this plugin, if you want to subscribe registered users to you mailing lists on JustClick website.
PHP 5 is required.
Find more information on [JustClick website](http://www.justclick.ru/)

== Installation ==

1. Upload `justclick-subscriber` directory to the `/wp-content/plugins/` directory
1. Activate the JustClick registration plugin through the 'Plugins' menu in WordPress
1. New item will appear in the Options menu
1. Check "Subscribe user after register" and fill in JustClick subscribe form code on the plugin settings page
1. Try to register
